# Tools

## Frontend
- Node.JS
- Angular 20
- Bootstrap 5
- FontAwesome

## Backend
- ASP.NET 10
- MongoDB Community Server

# How to run
1. Install bools
2. Install NPM packages
3. Set backend server settings in .Net Backend/appsettings.json and .Net Backend/appsettings.Production.json
4. Set frontend server settings in Angular Frontend/package.json and  Angular Frontend/src/app/app-config.ts (backend server host must match in backend and frontend settings)
5. run .Net Backend/start-production.bat and Angular/start.bat
6. Angular client will be on http://localhost/ by default
